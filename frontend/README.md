# Frontend - Deopay (Flutter)

Carpeta para la aplicación móvil frontend implementada en Flutter.

Contenido inicial:
- `pubspec.yaml` - dependencias y assets
- `lib/main.dart` - punto de entrada de la app

Instrucciones rápidas:

1. Asegúrate de tener Flutter instalado y configurado (`flutter doctor`).
2. Desde esta carpeta ejecuta:

```bash
flutter pub get
flutter run
```

Si quieres que yo genere más pantallas o arte, dime qué componentes quieres primero.
